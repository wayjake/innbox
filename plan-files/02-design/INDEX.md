# 02 - Design System

"Sage Trust" design system — professional, calming, trustworthy.

## Files

| File | Description |
|------|-------------|
| `DESIGN-SYSTEM.md` | Colors, typography, component specs, Tailwind config |

## Quick Reference

### Colors
- Primary: `sage-500` (#5B7F6D)
- Accent: `forest` (#2D4F3E)
- Background: `cream` (#F8F9F6)

### Typography
- Headings: Libre Baskerville (serif)
- Body: Inter (sans-serif)

### Rules
- All transitions: 200ms ease-out
- Border radius: 6-8px (never pill)
- No separate CSS files
